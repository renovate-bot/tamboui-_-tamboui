# TamboUI brand assets

Icons and logos for the TamboUI project. All assets are available as **SVG** (scalable) and **PNG** (raster at common sizes).

## Naming

- **`tamboui_icon_*`** — App icon / favicon (square).
- **`tamboui_logo_*`** — Full logo (wordmark + icon).
  - **`horizontal`** — Icon and text side by side (headers, nav, wide layouts).
  - **`vertical`** — Icon above text (splash, cards, narrow layouts).
- **Style**
  - **`default`** — Single-color (works on light or dark when inverted).
  - **`gradient`** — Gradient version for marketing and high-impact use.
